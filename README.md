# Lizzy Castrillo Portfolio Website

Personal portfolio website for social media influencer Lizzy Castrillo. Built with HTML, CSS, and vanilla JavaScript.

## Features

- Responsive design
- Mobile-first approach
- Modern, minimalist aesthetic
- Portfolio gallery
- Collaboration packages
- Booking integration with Cal.com

## Technologies

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- Google Fonts (Montserrat)

## Setup

1. Clone the repository
2. No build process required - static HTML site
3. Deploy to any static hosting service

## Development

To run locally:
```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`

## Deployment

The site is deployed using Cloudflare Pages for optimal performance and security.
